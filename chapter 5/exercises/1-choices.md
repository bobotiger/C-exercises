## 5
- 下列程序的输出结果是 ______。
```c
fun(int a, int b, int c)
{
    c = a * b;
}

int main(void)
{
    int c;
    fun(2, 3, c);
    printf("%d\n", c);
    return 0;
}

A. 2  B. 3  C. 6  D. 无法确定
```
### 解析
- 这道题其实会让人很困惑
  - 我揣测出题人，期望你掌握：
    - C语言中，如果不对局部变量进行初始化，那么变量的值就是一个不确定的值
    - 函数调用时，实参将值复制一份给形参，形参的改变不会影响实参的值
      - 例如：这里实参c的值被复制到形参c中，形参c被改变为6，但是函数退出后，打印的实参c，由于实参c并没有初始化，因此是一个不确定的值；
- 然而，在执行以上程序时，本程序却会输出0，这就有点诡异了；
- 另外，在早期的C语言标准中，规定不给函数指定返回值类型，会默认视为int类型，然而，在C99以后，这个特性就被废弃了，因此导致执行以上程序时，编译器报语法错误；

- 参考答案
```
D
```

## 6
- 以下程序的运行结果是 ______。
```c
int x = 5, y = 6;
void incxy()
{
    x++;
    y++;
}

int main(void)
{
    int x = 3;
    incxy();
    printf("%d,%d\n", x, y);
    return 0;
}

```
```
A. 3,6  B. 4,7  C. 3,7  D. 6,7
```
### 解析
- 本程序中定义了两个全局变量 x 和 y
- 每次调用无参，无返回值的自定义函数incxy，会对全局变量x和y分别进行+1操作
- main函数中
  - 定义了一个局部变量x，其与全局变量x同名，那么在你使用变量x时，编译器会优先使用局部变量x
  - 调用incxy，对全局变量x和y
  - 打印x，y，其中x就是局部变量x, y是全局变量，此时已经在原有值上加了1；
    - 所以应输出：3,7

- 答案
```
C
```
