# 练习 5-3
- 字符金字塔：输入一个正整数 n 和一个字符 ch，输出 n 行由字符 ch 构成的字符金字塔。试编写相应程序

- 程序实现
```c
#include <stdio.h>

// 打印字符金字塔
void print_pyramid(int n, char ch);

int main() {
    int n;
    char ch;

    printf("请输入一个正整数 n: ");
    scanf("%d", &n);  // 输入行数
    printf("请输入一个字符 ch: ");
    scanf(" %c", &ch); // 注意前面加空格以忽略缓冲中的换行符

    print_pyramid(n, ch);

    return 0;
}

void print_pyramid(int n, char ch) {
    for (int i = 1; i <= n; i++) {
        // 打印前导空格
        for (int j = 0; j < n - i; j++) {
            printf(" ");
        }
        // 打印字符
        for (int j = 0; j < (2 * i - 1); j++) {
            printf("%c", ch);
        }
        printf("\n");
    }
}
```
