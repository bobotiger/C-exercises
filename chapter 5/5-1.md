# 练习 5-1
- 使用函数求 1 到 n 之和：输入一个正整数 n，输出 1~n 之和。要求定义和调用函数 sum(n) 求 1~n 之和。若要计算 m~n (m<n) 之和，又该如何定义函数？试编写相应程序。

## 求 1 到 n 之和的函数
- 只需要一个参数n
```c
#include <stdio.h>
// 计算 1 ~ n 之和
int sum(int n);
// 计算 m ~ n 之和
int sum_range(int m, int n);

int main() {
    int m, n;

    // 第一部分：输入 n，计算 1~n 之和
    printf("请输入一个正整数 n: ");
    scanf("%d", &n);
    printf("1 到 %d 的和为: %d\n", n, sum(n));

    // 第二部分：输入 m, n，计算 m~n 之和
    printf("请输入两个正整数 m 和 n (m < n): ");
    scanf("%d %d", &m, &n);
    printf("%d 到 %d 的和为: %d\n", m, n, sum_range(m, n));

    return 0;
}


int sum(int n) {
    int total = 0;
    for (int i = 1; i <= n; i++) {
        total += i;
    }
    return total;
}


int sum_range(int m, int n) {
    int total = 0;
    for (int i = m; i <= n; i++) {
        total += i;
    }
    return total;
}
```
