# 练习 5-2
- 使用函数求最大值：输入 2 个数，输出其中较大的数。要求定义和调用函数 max(a, b) 找出并返回 a、b 中较大的数。试编写相应程序。

- 程序实现
```c
#include <stdio.h>
// 定义函数 max，比较两个数并返回较大的一个
int max(int a, int b);


int main() {
    int num1, num2;

    // 输入两个整数
    printf("请输入两个整数: ");
    scanf("%d %d", &num1, &num2);

    // 调用 max 函数并输出结果
    printf("较大的数是: %d\n", max(num1, num2));

    return 0;
}

int max(int a, int b) {
    if (a > b)
        return a;
    else
        return b;
}
```

- 执行结果
```
请输入两个整数: 312 332
较大的数是: 332
```
