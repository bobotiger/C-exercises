# 练习 3-1
- 例3-4中使用`else-if`语句求解多分段函数，为了检查`else-if`语句的三个分支是否正确，已经设计了三组测试用例，请问还需要增加测试用例吗？为什么？如果要增加，请给出具体的测试用例并运行程序。 

- 例3-4
```c
#include <stdio.h>
int main(void){
    double x,y;
    printf("Enter x:");
    scanf("%lf", &x);
    if(x<0){
        y = 0;
    }else if(x<=15){
        y = 4 * x / 3;
    }else{
        y = 2.5 * x - 10.5;
    }
    printf("f(%.2f) = %.2f\n", x, y);
    return 0;
}
```
- 测试用例
```
Enter x:-0.5
f(-0.50) = 0.00

Enter x:9.5
f(9.50) = 12.67

Enter x:21.3
f(21.30) = 42.75
```
- 解答
```
分段函数的3个取值区间为
    x<0
    0<=x && x<=15
    x>15
例3-4中已经提供的3组测试用例，覆盖了以上3个取值区间，但对于两个临界点0 和 15 没有进行测试
因此需要添加以下两组测试用例
Enter x:0
f(0.00) = 0.00

Enter x:15
f(15.00) = 20.00
经测试，程序执行符合需求
```
