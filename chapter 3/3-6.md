# 练习 3-6 
- 在例3-8程序中，如果把switch语句中所有的break都去掉，运行结果会改变吗？如果有变化，输出什么？为什么？

- 例 3-8
```c
#include <stdio.h>
/*查询自动售货机中的商品价格*/
int main(void){
    int choice,i;
    double price;

    printf("[1] Select crisps\n");
    printf("[2] Select popcorn\n");
    printf("[3] Select chocolate\n");
    printf("[4] Select cola\n");
    printf("[0] exit\n");
    for(int i=1; i<=5; i++){
        printf("Enter choice:");
        scanf("%d",&choice);
        if(choice==0){
            break;
        }
        switch(choice){
            case 1: price=3.0; break;
            case 2: price=2.5; break;
            case 3: price=4.0; break;
            case 4: price=3.5; break;
            default: price=0.0; break;
        }
        printf("Price=%.1f\n",price);
    }
    printf("Thanks\n");    
    return 0;
}
```
- 执行结果
```
[1] Select crisps
[2] Select popcorn
[3] Select chocolate
[4] Select cola
[0] exit
Enter choice:1
Price=3.0
Enter choice:2
Price=2.5
Enter choice:3
Price=4.0
Enter choice:4
Price=3.5
Enter choice:0
Thanks
```
- 将所有`case`分支的`break`语句全部删除后的程序执行结果
```
[1] Select crisps
[2] Select popcorn
[3] Select chocolate
[4] Select cola
[0] exit
Enter choice:1
Price=0.0
Enter choice:2
Price=0.0
Enter choice:3
Price=0.0
Enter choice:4
Price=0.0
Enter choice:0
Thanks
```
- 可见程序执行结果发生了改变，任何选择的商品价格都为0；
- 此时，switch中的每一个case分支都会被执行，在对同一变量的多个赋值操作顺序执行后， 该变量中的值由最后一个赋值操作决定，因此最终price变量中的值为0；
