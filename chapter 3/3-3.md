# 练习 3-3
- 统计学生平均成绩与及格人数：输入一个正整数n，再输入n个学生的成绩，计算平均成绩，并统计所有及格学生的人数。试编写相应程序。

- 解答
```
    输入学生人数n，以及n个学生的成绩 s1 s2 s3 ... sn
    计算平均成绩：(s1+s2+s3+...+sn)/n
    统计及格学生人数：
        int count = 0;
        for(int i=0; i<n; i++){
            if(s[i] >= 60){
                count += 1;
            }
        }
```
```c
#include <stdio.h>
int main(void){
    int n;
    printf("Enter number of students: ");
    scanf("%d", &n);

    double score;
    double sum = 0.0;
    int pass_count = 0;

    for(int i = 0; i < n; i++){
        printf("Enter score for student %d: ", i + 1);
        scanf("%lf", &score);
        sum += score;
        if(score >= 60){
            pass_count++;
        }
    }

    double average = sum / n;
    printf("Average score: %.2f\n", average);
    printf("Number of passing students: %d\n", pass_count);

    return 0;
}
```
