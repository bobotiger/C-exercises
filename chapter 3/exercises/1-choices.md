# 一 选择题
## 1
- 有一函数：
$$
y = 
\begin{cases} 
1, & x > 0 \\ 
0, & x = 0 \\ 
-1, & x < 0
\end{cases}
$$
以下程序段中错误的是______。

```
A. if(x > 0) y = 1;
   else if(x == 0) y = 0;
   else y = -1;

B. y = 0;
   if(x > 0) y = 1;
   else if(x < 0) y = -1;

C. y = 0;
   if(x >= 0);
   if(x > 0) y = 1;
   else y = -1;

D. if(x >= 0)
       if(x > 0) y = 1;
       else y = 0;
   else y = -1;
```

### 答案
```
C
```
- 选项A：
```c
if(x > 0) 
    y = 1;
else if(x == 0) 
        y = 0;
     else 
        y = -1;
```
- 逻辑流程
```
if (x > 0)       // 覆盖 x > 0 的情况
    ├─ y = 1
else
    if (x == 0)  // 覆盖 x == 0 的情况
     └─ y = 0;
    else         // 覆盖 x < 0 的情况
     └─ y = -1;
```

- 选项B：
```c
y = 0;
if(x > 0) 
    y = 1;
else if(x < 0) 
    y = -1;
```
- 逻辑流程
```
y = 0;          // 覆盖 x == 0 的情况
if (x > 0)      // 覆盖 x > 0 的情况
    ├─ y = 1
else           
    if (x < 0)  // 覆盖 x < 0 的情况
     └─ y = -1;
```

- 选项C：
```c
y = 0;
if(x >= 0)
    ;
if(x > 0) 
    y = 1;
else 
    y = -1;
```
- 逻辑流程
```
y = 0;          
if (x >= 0)     // 因为包含空语句，无有效代码，因此为多余的if语句；
  └─;
if (x > 0)      // 覆盖 x > 0 的情况
  └─ y = 1;
else            // 覆盖 x <= 0 的情况, 这不符合题意
  └─ y = -1;
```

- 选项D：
```c
if(x >= 0)
    if(x > 0) 
        y = 1;
    else 
        y = 0;
else y = -1;
```
- 逻辑流程
```          
if (x >= 0)         // 覆盖 x >= 0 的情况；
    if (x > 0)      // 覆盖 x > 0 的情况
      └─ y = 1;
    else            // 覆盖 x == 0 的情况
      └─ y = 0;
else                // 覆盖 x < 0 的情况
  └─ y = -1;
```

## 4
- 能正确表示逻辑关系 “a >= 10 或 a <= 0” 的 C 语言表达式是______。  
```
A. a >= 10 or a <= 0  
B. a >= 0 | a <= 10  
C. a >= 10 && a <= 0  
D. a >= 10 || a <= 0  
```
### 答案
```
D
```
- `||` 是逻辑或，`&&` 是逻辑与，`or` 不可直接用，`|` 是按位或，不是逻辑或

A. break 语句只能用于 switch 语句  
B. 在 switch 语句中必须使用 default

D. 在 switch 中，不一定使用 break 语句

A. 和之前与其具有相同缩进位置的 if 配对  
B. 和之前与其最近的 if 配对

## 8
- 在执行以下程序段时，为使输出结果为 t = 4，则给 a 和 b 输入的值应满足的条件是______。
```c
int a, b, s, t;
scanf("%d,%d", &a, &b);

s = 1; t = 1;
if(a > 0) s = s + 1;
if(a > b) t = s + t;
else if(a == b) t = 5;
else t = 2 * s;
printf("t = %d\n", t);
```
```
A. a > b  
B. 0 < a < b  
C. 0 > a > b  
D. a < b < 0
```
### 答案
```
B
```
- 分析 
```
若 a > b：
    分支：t = s + t = s + 1
    要 t = 4 → s + 1 = 4 → s = 3
    s 初始为 1，只有 a > 0 时 s 才会加 1，所以 a > 0 时 s = 2，不可能等于 3。因此 a > b 的情况不成立。
若 a == b：
    分支：t = 5
    与要求不符（我们要 t = 4），排除。
    若 a < b：
分支：t = 2 * s
    要 t = 4 → 2 * s = 4 → s = 2
    s 为 2 只有在 a > 0 时成立，因此必须 a > 0 且 a < b。
```
- 当 `a > 0` 时 `s` 变为 `2`，再由 `a < b` 进入 `t = 2 * s` 分支，得到 `t = 4`。
