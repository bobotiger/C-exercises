## 练习 3-4
- 统计字符：输入一个正整数n, 再输入n个字符，统计其中英文字母、空格或回车、数字字符和其他字符的个数，试编写相应程序。

- 解析
```
回车('\n') : 13(D) 0D(H)
空格(' ') ：32(D) 20(H) 
数字('0'-'9') ：48(D)-57(D) 30(H)-39(H)
大写英文字母('A'-'Z') ：65(D)-90(D) 41(H)-5A(H)
小写英文字母('a'-'z') ：97(D)-122(D) 61(H)-7A(H)

输入字符个数n，并输入n个字符c
if(c=='\n' || c==' '){
    回车符或空格计数
}else if(c>='0' && c<='9'){
    数字计数
}else if((c>='A' && c<='Z') || (c>='a' && c<='z')){
    英文字母计数
}else{
    其他字符计数
}
```
```c
#include <stdio.h>
/*统计字符：输入一个正整数n, 再输入n个字符，统计其中英文字母、
空格或回车、数字字符和其他字符的个数，试编写相应程序。*/
int main(void){
    int n;
    int letters = 0, spaces = 0, digits = 0, others = 0;
    char ch;
    printf("请输入一个正整数n: ");
    scanf("%d", &n);
    printf("请输入%d个字符:\n", n);
    // 清除输入缓冲区的换行符
    getchar();
    for(int i = 0; i < n; i++){
        ch = getchar();
        if((ch >= 'A' && ch <= 'Z') || (ch >= 'a' && ch <= 'z')){
            letters++;
        } else if(ch == ' ' || ch == '\n'){
            spaces++;
        } else if(ch >= '0' && ch <= '9'){
            digits++;
        } else{
            others++;
        }
    }
    printf("英文字母个数: %d\n", letters);
    printf("空格或回车个数: %d\n", spaces);
    printf("数字字符个数: %d\n", digits); 
    printf("其他字符个数: %d\n", others);
    return 0;
}
```
