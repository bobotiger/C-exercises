# 练习 3-8
- 查询水果的单价：有 4 种水果，苹果（apples）、梨（pears）、橘子（oranges）和葡萄（grapes），单价分别是 3.00 元/千克、2.50 元/千克、4.10 元/千克和 10.20 元/千克。在屏幕上显示以下菜单（编号和选项），用户可以连续查询水果的单价，当查询次数超过 5 次时，自动退出查询；不到 5 次时，用户可以选择退出。当用户输入编号 1~4，显示相应水果的单价（保留一位小数）；输入 0，退出查询；输入其他编号，显示价格为 0。试编写相应程序。
```
[1] apples
[2] pears
[3] oranges
[4] grapes
[0] Exit
```

```c
#include <stdio.h>
int main(void){
    double apples_price = 3.0;
    double pears_price = 2.5;
    double oranges_price = 4.1;
    double grapes_price = 10.2;
    int choice;
    int count = 0;
    printf("[1] apples\n[2] pears\n[3] oranges\n[4] grapes\n[0] Exit\n");

    while(count < 5){
        printf("Please enter your choice (0-4): ");
        scanf("%d", &choice);
        
        if(choice == 0){
            break;
        }

        switch(choice){
            case 1:
                printf("The price of apples is %.1f (yuan/kg)\n", apples_price);
                break;
            case 2:
                printf("The price of pears is %.1f (yuan/kg)\n", pears_price);
                break;
            case 3:
                printf("The price of oranges is %.1f (yuan/kg)\n", oranges_price);
                break;
            case 4:
                printf("The price of grapes is %.1f (yuan/kg)\n", grapes_price);
                break;
            case 0:
                printf("exit\n");
                break;
            default:
                printf("Please enter a valid choice.\n");
        }
        count++;
    }

    if(count == 5){
        printf("You have reached the maximum number of queries.\n");
    }else{
        printf("You have exited the query.\n");
    }

    return 0;
}
```
