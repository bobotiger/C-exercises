# 练习 3-10
- 改写下列if语句，使else和第一个if配对
```c
if(x<2)
    if(x<1) y=x+1;
    else y=x+2;
```
- 在C语言中，else 总是和最近的、尚未与其他 else 配对的 if进行匹配，这就是所谓的 **就近匹配原则**
- 如果有嵌套的多层`if`，而且中间没有使用花括号`{}`来明确代码块范围，`else`会与距离它最近并且没有`else`的那个`if`配对。使用花括号`{}`可以人为控制代码块范围，避免歧义;

## 方法1：使用大括号
```c
if(x<2){
    if(x<1) y=x+1;
}
else y=x+2;
```
- 以上语句的执行逻辑流程
```
if (x < 2)
    ├─ if (x < 1)
    │     └─ y = x + 1
    └─ [内层不成立时，不做任何事]
else
    └─ y = x + 2
```

## 方法2：
```c
if(x<2)
    if(x<1) y=x+1;
    else;
else y=x+2;
```
- 以上语句的执行逻辑流程
```
if (x < 2)
   ├─ if (x < 1)
   │     └─ y = x + 1
   │  else
   │     └─ [空语句，不做任何事]
else
   └─ y = x + 2

```
