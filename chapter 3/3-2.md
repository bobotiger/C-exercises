# 练习 3-2
- 计算符号函数的值：输入一个整数x, 计算并输出下列分段函数 `sign(x)` 的值。试编写写相应程序。
$$y=sign(x)=
\begin{cases}
-1,\quad x < 0\\
0, \quad x = 0\\
1, \quad x > 0
\end{cases}
\tag{1}$$

```c
#include <stdio.h>
int main(void) {
    int n;
    scanf("%d", &n);
    if(n<0){
        printf("sign(%d) = -1\n", n);
    }else if(n==0){
        printf("sign(%d) = 0\n", n);
    }else{
        printf("sign(%d) = 1\n", n);
    }
    return 0;
}
```
