# 练习 3-5
- 输出闰年：输出21世纪中截至某个年份之前的所有闰年年份。判断闰年的条件是：能被4整除但不能被100整除，或者能被400整除。试编写相应程序。

```c
#include <stdio.h>
int main(void){
    int year, end_year;

    printf("请输入一个年份（2001-2100之间）：");
    scanf("%d", &end_year);
    
    if(end_year < 2001 || end_year > 2100){
        printf("输入年份不在范围内！\n");
        return 1;
    }

    printf("21世纪截至%d年前的闰年有：\n", end_year);
    for(year = 2001; year < end_year; year++){
        if((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)){
            printf("%d\n", year);
        }
    }    
    return 0;
}
```
