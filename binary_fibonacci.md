# 题目描述：
- 二维斐波那契波动数列满足以下递推式：
$$
\begin{cases}
a_{i,j} = 1, & (i,j \in \{1\}) \\[4pt]
a_{i,j} = a_{i-1, j}, & (2 \le i, \; j \in \{1\}) \\[4pt]
a_{i,j} = a_{i, j-1}, & (2 \le j, \; i \in \{1\}) \\[4pt]
a_{i,j} = a_{i-1, j} + a_{i, j-1}, & (2 \le i, j)
\end{cases}
$$

- 给定正整数 $n, m$，求 $a_{n,m}$ 的值。由于结果可能很大，请输出 $a_{n,m}$ 对 $10^9 + 7$ 取模后的结果。

## 输入描述 
- 在一行中输入两个正整数 $n, m$ $(1 \le n, m \le 10^3)$，分别表示行下标和列下标。

## 输出描述  
- 输出一个整数，表示 $a_{n,m} \bmod (10^9 + 7)$ 的值。

# 解析
## 问题与输入/输出
- 输入：正整数 $n, m (1 ≤ n, m ≤ 1000)$。
- 输出：按题中递推式定义的 $a_{n,m}$ 对 $10^9+7$ 取模的值。
- 正常成功条件：返回非负整数 $< 10^9+7$。
- 错误模式：输入不合法（题目里不考虑），n 或 m 极端值（1）是合理情况。
## 数学解析（组合数推导）
- 递推：
- $a_{1,1} = 1$；第一行、第一列全为 1；
- 对于 $i,j ≥ 2$：$a_{i,j} = a_{i-1,j} + a_{i,j-1}$。
- 把问题看成从格子 $(1,1)$ 到 $(n,m)$ 的路径计数：每一步只能向下（增 i）或向右（增 j）。要到 $(n,m)$ 必须走 $(n-1)$ 次向下和 $(m-1)$ 次向右，总步数为 $(n+m-2)$。在这些步中选择哪 $(n-1)$ 步是向下（或等价选择哪 $(m-1)$ 步是向右），因此
- $a_{n,m} = C(n+m-2, n-1) = C(n+m-2, m-1)$.
该公式可通过组合解释或用归纳法证明，并满足题的边界条件（当 $n=1$ 或 $m=1$ 时 $C(...,0)=1）$。

## 算法选择与复杂度

直接动态规划（DP）
思路：按行或按列计算 a_{i,j}，转移 a[i][j] = a[i-1][j] + a[i][j-1]（取模）。
时间：O(n*m)。空间可降为 O(min(n,m))（使用一维滚动数组）。
适用场景：当 n,m 较小或不想写组合数代码时，简单可靠。
组合数 + 预处理（推荐）
思路：利用 a_{n,m} = C(n+m-2, n-1)。在模为质数（MOD=1e9+7）下，使用阶乘与逆元预处理：
预计算 fact[k] (k=0..n+m-2) 和 invfact[k]（用快速幂与费马小定理求逆）。
结果 = fact[n+m-2] * invfact[n-1] % MOD * invfact[m-1] % MOD。
时间：预处理 O(n+m), 单次查询 O(1)；空间 O(n+m)。
适用场景：当需要多次查询或想更快求解时优先使用。由于 n+m ≤ 2000，预处理很快且安全。
边界与注意事项

当 n=1 或 m=1 时，返回 1。
模数 MOD=1e9+7 为质数 → 可使用费马小定理计算模逆。
如果不想预分配数组，可用乘法累乘/约分的组合数乘法法（但需要注意中间除法/逆元），因为上界不是很大，阶乘方法更清晰、稳健。
