## 1
- 假定int类型变量占用两个字节，则以下定义的数组a在内存中所占字节数是 ________。
```c
int a[10] = {0, 2, 4};
```
```
A. 20
B. 10
C. 6
D. 3
```
- 答案
```
A
```


## 3
- 以下程序段的输出结果是 ________。
```c
int aa[4][4] = {
    {1, 2, 3, 4},
    {5, 6, 7, 8},
    {3, 9, 10, 2},
    {4, 2, 9, 6}
};
int i, s = 0;

for(i = 0; i < 4; i++)
    s += aa[i][3];

printf("%d\n", s);
```
```
A. 11
B. 19
C. 13
D. 20
```
- 答案
```
D

将二维数组aa中的每行第4列累加到s中
s = 4 + 8 + 2 + 6 = 20
```
