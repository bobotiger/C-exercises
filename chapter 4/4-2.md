# 练习 4-2
- 运行例 4-2 程序时，如果将最后一个输入数据改为 -2，运行结果有变化吗？如果第一个输入数据是 -1，运行结果是什么？为什么？

## 例4-2
```c
/* 输入一批学生的成绩，以负数作为结束标志，计算平均成绩，并统计不及格人数 */
#include <stdio.h>
int main(void)
{
    int count, num;              /* num 记录输入的个数，count 记录不及格人数 */
    double score, total;         /* 分别存放成绩、成绩之和 */

    num = 0;
    total = 0;
    count = 0;
    printf("Enter scores:");     /* 输入提示 */
    scanf("%lf", &score);        /* 输入第一个数据，%lf 中的 l 是字母 */

    /* 当输入数据 score 大于等于 0 时，执行循环 */
    while(score >= 0)
    {
        total = total + score;   /* 累加成绩 */
        num++;                   /* 计数 */
        if(score < 60)
            count++;
        scanf("%lf", &score);    /* 读入一个新数据，为下次循环做准备 */
    }

    if(num != 0)
    {
        printf("Average is %.2f\n", total / num);
        printf("Number of failures is %d\n", count);
    }
    else
    {
        printf("Average is 0\n");
    }
    return 0;
}
```
### 执行结果
```
Enter scores:76 88 73 54 82 -1
Average is 74.60
Number of failures is 1
```
### 将最后一个输入数据改为-2
```
Enter scores:76 88 73 54 82 -2
Average is 74.60
Number of failures is 1
```
- 可见结果没有什么变化

### 如果第一个输入数据是 -1
```
Enter scores:-1 76 88 73 54 82      
Average is 0
```
- 第一个分数就设置为负数，会导致循环一次都不执行
