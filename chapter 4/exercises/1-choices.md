## 1
- 以下程序段 ______ 不能完成求 s = 1 + 2 + … + n − 1。

A.
```c 
int i, n, s = 0;  
scanf("%d", &n);  
for(i = 1; i < n; i++)  
    s = s + i;  
```
B.
```c 
int i, n, s = 0;  
scanf("%d", &n);  
for(i = 1; i <= n-1; ++i)  
    s = s + i;    
```
C.
```c 
int i, n, s = 0;  
scanf("%d", &n);  
for(i=n-1; i > 0; i--)  
    s = s + i;   
```
D.
```c 
int i, n, s = 0;  
scanf("%d", &n);  
for(i=n-1; i > 0; ++i)  
    s = s + i;  
```
### 解析
```
A. 循环从1到n-1，步长为1(i++)，s初始值为0，每次循环执行 s = s + i，显然可以完成求和任务
B. 循环从1到n-1，步长为1(++i)，s初始值为0，每次循环执行 s = s + i，显然可以完成求和任务
C. 循环从n-1到1，步长为1(i--)，s初始值为0，每次循环执行 s = s + i，显然可以完成求和任务
D. 循环从n-1到无穷，步长为1(++i)，s初始值为0，每次循环执行 s = s + i，由于i>0永远满足，循环无法结束
```
## 2
- 输入 65␣14<Enter>，以下程序段的输出结果为 ______
```c
int m, n;  
scanf("%d%d", &m, &n);  
while(m!=n){
    while(m>n)m=m-n;
    while(n>m)n=n-m;
}  
printf("m=%d\n", m);  
```
```
A. m = 3
B. m = 2
C. m = 1
D. m = 0
```
### 解析
- 外层循环条件：`m!=n`, 此时 m=65，n=14，条件为真
  - 内层循环1：`m>n`，条件为真 则执行 `m=m-n`，直到`m<=n`  65 - 14 = 51 - 14 = 37 - 14 = 23 - 14 = 9 
  - 内层循环2：`n>m`, 条件为真 则执行 `n=n-m`, 直到`n<=m`  14 - 9 =  5
- 外层循环条件：`m!=n`, 此时 m=9，n=5，条件为真
  - 内层循环1：`m>n`，条件为真 则执行 `m=m-n`，直到`m<=n`  9 - 5 = 4 
  - 内层循环2：`n>m`, 条件为真 则执行 `n=n-m`, 直到`n<=m`  5 - 4 = 1 
- 外层循环条件：`m!=n`, 此时 m=4，n=1，条件为真
  - 内层循环1：`m>n`，条件为真 则执行 `m=m-n`，直到`m<=n`  4 - 1 = 3 - 1 = 2 - 1  = 1 
  - 内层循环2：`n>m`, 此时 m=1, n=1 条件为假
- 外层循环条件：`m!=n`, 此时 m=1，n=1，条件为假，循环退出

- 下列程序段的输出结果是 ______
```c
int i;  
for(i = 1; i < 6; i++){
    if(i % 2 != 0) {  
        printf("#");  
        continue;  
    }  
    printf(" * ");  
}
```
```
A. # * # * #
B. #####
C. * * * * *
D. * # * # *
```
### 解析
- 循环从i=1 到 i=5，共循环5次
- 第一次循环时，i=1，`i%2!=0` 条件为真，那么打印"#", 然后continue，跳过本次循环continue之后的语句，即不会打印" * "
- 第二次循环时，i=2，`i%2!=0` 条件为假，if语句块跳过，打印" * "
- 第三次循环时，i=3，`i%2!=0` 条件为真，那么打印"#", 然后continue，跳过本次循环continue之后的语句，即不会打印" * "
- 第四次循环时，i=4，`i%2!=0` 条件为假，if语句块跳过，打印" * "
- 第五次循环时，i=5，`i%2!=0` 条件为真，那么打印"#", 然后continue，跳过本次循环continue之后的语句，即不会打印" * "
- 循环结束
