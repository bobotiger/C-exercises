# 练习 4-3
- 输入一个正实数 eps，计算序列 1 − 1/4 + 1/7 − 1/10 + 1/13 − 1/16 + … 的值，精确到最后一项的绝对值小于 eps（保留 6 位小数）。试编写相应程序。

## 思路
- 初始状态：
```c
double eps = 0.000001;
double sum = 0.0;
int i = 0;
int sign = 1;
double item;
```
- 循环不变式
```c
item = sign * 1.0/(1+3*i);
sum += item;
```

- 循环终止条件
```c
fabs(item) < eps
```
- 注意：
> 如果item初始时值未确定，那么需要使用do...while循环

> 如果item初始化为1，那么也可以使用while循环

## 实现
- item初始化为1时
```c
#include <stdio.h>
#include <math.h>
int main(void)
{
    double eps;
    int sign = 1;
    double term = 1;
    double sum = 0.0;
    printf("请输入一个正实数 eps：");
    scanf("%lf", &eps);
    int i = 0;
    while (fabs(term) >= eps){
        term = sign * 1.0 / (1 + 3 * i);
        sum += term;
        sign = -sign;
        i++;
    } 
    printf("序列的和为：%.6f\n", sum);

    return 0;
}
```
- item未初始化时
```c
#include <stdio.h>
#include <math.h>
int main(void)
{
    double eps;
    int sign = 1;
    double term;
    double sum = 0.0;
    printf("请输入一个正实数 eps：");
    scanf("%lf", &eps);
    int i = 0;
    do {
        term = sign * 1.0 / (1 + 3 * i);
        sum += term;
        sign = -sign;
        i++;
    } while (fabs(term) >= eps); 
    printf("序列的和为：%.6f\n", sum);

    return 0;
}
```
