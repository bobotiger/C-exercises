# 练习 2-12
- 输出华氏-摄氏温度转换表：输入两个整数 lower 和 upper，输出一张华氏-摄氏温度转换表，华氏温度的取值范围是 `[lower, upper]`, 每次增加 $2^∘F$, 计算公式同例2-6。试编写相应程序。

## 思路
- 输入两个整数 lower 和 upper
- 将 `[lower, upper]` 取值范围按 步长 $2^∘F$ 取值，
  - 进行华氏-摄氏温度转换计算
  - 输出华氏及转换后的摄氏温度

## 编写程序
```c
#include <stdio.h>
int main(void){
    
    int lower;
    int upper;
    printf("Please enter the lower and upper bounds(Fahrenheit): ");
    scanf("%d %d", &lower, &upper);
    
    printf("Fahrenheit \tCelsius\n");
    for(int i=lower; i<=upper; i+=2){
        printf("%10d\t%7d\n", i, (int)((5.0/9.0)*(i-32)));
    }    
    return 0;
}
```
