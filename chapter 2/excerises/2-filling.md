# 二 填空题
## 1
- 假设 k 是整型变量，计算表达式 1/k，结果的数据类型是________, 计算表达式1.0/K， 结果是_______。

```
答案：
    int  
    double
解析：
    1 为 int 类型常量，k 是整型变量，因此 1/k 的整除结果类型为 int；
    1.0 为 double 类型常量，k 是整型变量，因此 1.0/k 的小数除法结果类型为 double ；
```
## 2
- 输入 3 和 2，下列程序段的输出结果是________。
```c
int a, b;
scanf("%d%d", &a, &b);
a=a+b;
b=a-b;
a=a-b;
printf("a=%d#b=%d\n",a,b);
```
```
答案：
    a=2#b=3   

解析：
    a = 3  b = 2
    a=a+b --> a = 5
    b=a-b --> b = 3
    a=a-b --> a = 2
    因此输出结果为：a=2#b=3
```
## 3
- 变换变量的值。输入 a 和 b, 然后交换它们的值， 并输入交换后 a 和 b 的值。请填空。
```c
int a,b,temp;
printf("Enter a, b:");
scanf("%d%d", &a, &b);
________;
a=b;
________;
printf("a=%d#b=%d\n",a,b);
```

```
答案：
    temp=a
    b=temp
解析：
    temp 用于暂存之后需要使用的值
    第一空后为 a=b，变量a 的原值会被变量b 的值覆盖，因此需将变量a 的值先暂存；
    最后将 temp 中暂存的之前变量 a 的值赋值给变量 b，完成 a 与 b 变量值的交换;
```

## 8
- 下列程序段的输出结果是_______。
```c
int k, flag;
if(k=0){
    flag=0;
}else{
    flag=1;
}
printf("k=%d#flag=%d\n", k, flag);
```
```
答案：
    k=0#flag=1
解析：
    k=0 中使用的是赋值运算符，表达式k=0的值为0；
    因此，条件语句执行条件为假的分支，即 flag=1
    最终输出 k=0#flag=1
```

## 10
- 下列程序段的输出结果是________。
```c
int i;
double s=0;
for(i=1; i<4; i++){
    s = s+1.0/i;
    printf("i=%d#s=%.3f\n", i, s);
}
printf("i=%d#s=%.3f\n", i, s);
```

```
答案：
    i=1#s=1.000
    i=2#s=1.500
    i=3#s=1.833
    i=4#s=1.833
```
